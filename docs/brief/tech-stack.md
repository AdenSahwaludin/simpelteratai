Saya pakai laravel 12 dan blade untuk membuat web aplikasi manajemen TK. Berikut adalah stack teknologi yang saya gunakan:

-   **Backend Framework**: Laravel 12
-   **Frontend Templating**: Blade
-   **Database**: MySQL
-   **Styling**: Tailwind CSS
-   **Routing**: Laravel's built-in routing system
-   **ORM**: Eloquent ORM
